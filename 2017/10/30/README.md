# 静态内部类的简单应用（一次遍历求最大最小值）

静态内部类是使用 static 修饰的内部类，在静态内部类中，可以使用外部类定义的静态域，但是不能使用非静态域。这是静态内部类与非静态内部类的重要区别。

定义一个简单的静态内部类的代码如下：

```java
public void book(){
    public static class JavaLearn{}
}
```

**注意：** 不要将 JavaLearn 类声明成 private 的，否则不能使用其中定义的方法。
